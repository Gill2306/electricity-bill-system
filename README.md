# Electricity Bill System in C

## Project Overview

This is a simple Electricity Bill System created using the C programming language.

The program calculates electricity bills based on unit consumption using a slab-based pricing system.

---

## Features

- Customer name input
- Customer ID input
- Electricity unit calculation
- Slab-based billing system
- Tax calculation
- Final bill generation

---

## Technologies Used

- C Programming Language
- GCC Compiler

---

## Billing Logic

| Units Range | Cost Per Unit |
|-------------|---------------|
| 0 - 100     | ₹2            |
| 101 - 200   | ₹5            |
| Above 200   | ₹8            |

---

## Tax

- 18% tax added to total bill
